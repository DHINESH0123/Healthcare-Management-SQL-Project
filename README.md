1. PROJECT OVERVIEW
This project focuses on analyzing a healthcare management system using
Microsoft Excel and Oracle SQL. The analysis covers patient visits,
doctor performance, treatment details, and billing operations.

2. DATA CLEANING
Initial data cleaning was performed using Microsoft Excel.
- Removed null and blank values
- Removed duplicate records
- Standardized date and text formats
- Exported cleaned data as CSV files

The cleaned data was then imported into Oracle for SQL analysis.

3. DATASET
The project uses five relational tables:
- PATIENTS
- DOCTORS
- APPOINTMENTS
- TREATMENTS
- BILLING

4. SQL CONCEPTS USED
- SELECT & FILTERING
- AGGREGATE FUNCTIONS
- GROUP BY & HAVING
- JOINS
- SUBQUERIES
- WINDOW FUNCTIONS
- BUSINESS ANALYSIS QUERIES

5. BUSINESS QUESTIONS ANSWERED
- Which doctors handle the most patients?
- Who are repeat patients?
- Which treatments are most expensive?
- What percentage of appointments are cancelled?
- Where does revenue leakage occur?

6. TOOLS USED
- MICROSOFT EXCEL (DATA CLEANING)
- ORACLE SQL (DATA ANALYSIS)
